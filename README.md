# Proyecto_Entornos_de_Desarrollo_Sergio-2
## Descripción
Este proyecto es una aplicación web simple desarrollada con HTML, CSS y JavaScript. Permite a los usuarios registrarse (Sign Up) e iniciar sesión (Login), para luego acceder a una página secundaria donde pueden explorar una tienda con diversos productos.  

El objetivo principal del proyecto es aprender a trabajar de manera más organizada y profesional con Git y GitHub, así como practicar la estructura de proyectos web y la navegación entre páginas.

## Características
- **Página de Login:** Permite a los usuarios iniciar sesión con credenciales previamente registradas.  
- **Página de Sign Up:** Permite a los nuevos usuarios registrarse en la plataforma.  
- **Tienda:** Una vez autenticado, el usuario es redirigido a una página donde se muestran varios productos disponibles.  
- **Aprendizaje de Git:** El proyecto se gestionó usando Git y GitHub para el control de versiones y colaboración.  

## Tecnologías Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Git / GitHub
